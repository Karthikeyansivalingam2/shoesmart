# 👟 ShoeSmart - Modern E-commerce Platform

ShoeSmart is a full-stack e-commerce application built specifically for a premium footwear shopping experience. Developed as a final year college project, this application demonstrates a robust implementation of the **MERN stack (MongoDB, Express, React, Node.js)** with advanced features like real-time admin dashboards and secure state management.

## 🚀 Key Features

### 🛍️ For Customers
*   **Futuristic UI/UX:** A high-performance, responsive interface built with React and Framer Motion for smooth transitions.
*   **Dynamic Product Catalog:** Real-time product browsing with category filters and instant search.
*   **Integrated Shopping Cart:** Persisted cart state across sessions for a seamless checkout experience.
*   **Personalized Dashboard:** Track real orders, manage multiple shipping addresses, and view purchase history.
*   **Secure Authentication:** JWT-based login/signup with role-based access control.

### 🛠️ For Administrators
*   **The Command Panel:** A state-of-the-art admin terminal to manage the entire ecosystem.
*   **Real-time Analytics:** Monitor total revenue, active orders, and customer growth instantly.
*   **Inventory Management:** Full CRUD operations to add, edit, or delete products and manage stock levels.
*   **Order Processing:** Interactive workflow to accept and process customer orders directly from the database.

## 💻 Tech Stack

*   **Frontend:** React.js, Tailwind CSS (Vanilla CSS components), Framer Motion, Lucide Icons.
*   **Backend:** Node.js, Express.js.
*   **Database:** MongoDB Atlas (Cloud Database).
*   **Security:** JSON Web Tokens (JWT), Bcrypt.js (Password Hashing).
*   **Payment Integration:** Razorpay (Support for UPI/Card/Wallet).

## 🛠️ Installation & Setup

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Karthikeyansivalingam2/Foot-wear.git
   ```

2. **Backend Setup:**
   * Navigate to the `server` folder.
   * Create a `.env` file with `MONGO_URI`, `JWT_SECRET`, and `PORT=5001`.
   * Run `npm install` and `npm start`.

3. **Frontend Setup:**
   * Navigate to the `client` folder.
   * Run `npm install` and `npm run dev`.
